# CSS BATTLE

CSS Battle é um jogo online divertido e desafiador onde você pratica CSS (Cascading Style Sheets) resolvendo desafios visuais. A ideia principal é reproduzir uma imagem-alvo usando o menor código CSS e HTML possível.
É como um jogo de lógica para quem gosta de programação front-end.

⚔️ Tem rivais ou modo competitivo?

Sim! O modo “Battle” permite:

  • Jogar desafios públicos.


  • Ver o ranking em tempo real (quem fez com menos código).


  • Subir no placar contra outros programadores do mundo inteiro.


  • Alguns jogadores criam rivalidades amigáveis e desafios entre amigos.
